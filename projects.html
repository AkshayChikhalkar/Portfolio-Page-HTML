Professional Projects

Project Name – TCU Validation

Role – EIS Software Developer

Description : Telematics Control Unit is an electronic device and an integral part of the modern-day vehicle. It takes care of all the ECUs and vehicle communication. TCU Validation Tool automates the process of testing TCU functionality and Server in real-time. It collects CAN, and Debug logs and validates with TSP through the MQTT protocol.

Responsibilities :

Responsible for analysis, design, implementation, and deployment of code.

Worked on ASE128 and HMAC encryption methods for TCU-TSP handshaking.

Implemented TCP/IP socket programming to access other simulators.

Implemented Web API call using MQTT protocol.

Designed and developed MySQL Server database.

Developed the installer for distributing the application across the users.

Environment : C#, WindowsForms, Visual Studio 2017, .Net Framework 4.5, MySQL Server 5.8, Windows 10, MQTT
